Detailed Description

template<unsigned int numInputs, unsigned int numOutputs>
class IndieNeat::Genotype< numInputs, numOutputs >::Gene

The class Gene contains all the information about a gene in the genome. With this class we can know where the gene is, between which Node. We also know its innovation number, which is the rank of the gene creation. Finally, we have the current coefficient of the gene, which is the knowledge contained inside it.
