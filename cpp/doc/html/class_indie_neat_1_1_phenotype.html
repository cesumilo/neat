Detailed Description

template<unsigned int numInputs, unsigned int numOutputs>
class IndieNeat::Phenotype< numInputs, numOutputs >

Allows to interpret Genotype into Phenotype.

With this class, you can create a phenotype based on the given genotype. It will allows you to compute the result of your model in terms of the inputs.

numInputsDescribe how many input takes the phenotype.
